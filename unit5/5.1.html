Lesson 5.1 Introduction to Memory

Memory-

Any indication that learning has occurred over time. The ability to store and retrieve information.


Spread of activation-

One idea leads to another.


Semantic memory-

Having similar memories with each other.


Episodic memories-

Memories unique to each person.


Flashbulb memory-

A unique and very emotional moment can create a strong and persistent memory that is subject to error.


Stages of memory-

Stages are encoding, storage, and retrieval in a sequential process. Your senses, like the eyes, process information towards areas like Wernicke’s and Broca’s area so that the brain can understand the information. Then you store the information in your brain. Retrieval is when active thinking occurs and an attempt to retrieve the information. All 3 of these processes must occur to remember and learn.


Information processing-

Atkinson Schiffrin created a three-stage model of memory that includes sensory memory, short-term memory, and long-term memory.


Sensory memory-

The initial momentary storage of info that lasts only for a second or two. Using our senses, it allows us to take in all information and determine if it is important enough to pay attention to. If it isn’t worth it, it is lost, decayed, or displaced.


Short term memory-

Memory that holds meaningful info for a short period, usually less than 30 seconds. It is meaningful because the sensory memory determined that it was important enough to pay attention to. It has a limited capacity of 7+/- 2 items. It can store more info if the info is grouped together.


Information will leave the short-term memory if it isn’t rehearsed or repeated. Repetition will keep the info in the short-term memory and is necessary for it to be transferred into long term memory.


Long term memory-

Memory that stores info on a relatively permanent basis. It appears to have a limitless capacity and info on long term memory is filed and coded so it can be retrieved when needed.


Some information skips the first two stages and enters long term memory immediately.


Selective attention-

Since we cannot focus on all the sensory info in the environment, we select info using attention.


Working memory-

Proposes that working memory acts as a workspace where info is retrieved and manipulated through rehearsal, instead of the simpler version of short-term memory. It is a central executive that contains auditory and visual processing that makes decisions and reasons. It allows us to briefly maintain info in an active state so we can do something with it. You have to work with the information before putting it into long-term memory, it can’t just sit somewhere like it would in short term memory.
